## 조건문

### if문
if문의 형태는 c++, java와 똑같은 형태이다.
```javascript
if(조건) {
  //동작
}
```

### switch문
switch문도 동일하다.
```javascript
switch (변수) {
  case 1:
    //동작
    break;
  case 2:
    //동작
    break;
  default:
    //동작
    break;
}
```


## 반복문

### for문
for문 역시 동일하다.
```javascript
//for
for(var i=0; i<array.length; i++) {
  //동작    
}

//for in
for (변수 in 배열) {
  //동작
}
```

### while문
while문 이하동문.
```javascript
while(조건) {
    //동작
}
```
